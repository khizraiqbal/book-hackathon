# Physical AI & Humanoid Robotics Book

This repository contains an educational book on Physical AI and Humanoid Robotics, built with Docusaurus.

## About

This educational resource covers:
- ROS 2 middleware fundamentals
- Python agent integration with ROS
- URDF for humanoid robots
- Practical examples and code snippets

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view the book in your browser.

## Modules

### Module 1 - The Robotic Nervous System (ROS 2)
- Chapter 1: ROS 2 Basics – Nodes, Topics, Services
- Chapter 2: Bridging Python Agents – Agent Patterns, rclpy Integration
- Chapter 3: URDF for Humanoids – Links, Joints, Sensors
